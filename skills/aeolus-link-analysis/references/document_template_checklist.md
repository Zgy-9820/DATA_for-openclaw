# 文档模板检查清单

用于在真正输出飞书文档前做快速自检。

## 必查章节
- 1. 查询身份
- 2. 一句话结论
- 3. 维度 SQL 逻辑表
- 4. 指标 SQL 逻辑表
- 5. 筛选项 SQL 逻辑表
- 6. 关键派生字段递归展开
- 7. 合并后的 WHERE 逻辑
- 8. 原始 SQL / 伪 SQL
- 9. 结果值
- 10. 风险点

## 必查格式
- 第 3 节是表格
- 第 4 节是表格
- 第 5 节是表格
- 第 9 节优先表格
- 第 8.1 节保留原始 SQL 骨架
- 第 8.2 节补可读版伪 SQL

## 必查内容
- 是否写明原始链接
- 是否写明真实 queryHistoryId
- 是否说明关键筛选条件
- 是否说明指标分子 / 分母
- 是否指出派生字段与口径风险
