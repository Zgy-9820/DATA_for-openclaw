# 风神技术方案生成 Skill

## Skill 说明
这是一个专门用于生成飞书Bubble目标管理技术方案的AI助手能力，能够根据截图和参考文档生成完整的技术方案文档。

## 使用场景
当需要为Bubble目标管理业务线接入新的指标时，使用此skill可以快速生成：
- 完整的技术方案文档（飞书文档格式）
- 详细的SQL实现代码
- 数据接入流程和配置说明
- 质量保障和风险评估
- 实施计划和责任人分工

## 输入要求
使用此skill需要提供以下信息：

### 1. 必需输入
- **截图**：包含待接入指标信息的表格图片（支持jpg/png等格式）
- **参考文档链接**：已有实现的飞书文档链接，用于参考格式和技术实现方式

### 2. 可选输入
- **业务方信息**：@业务方负责人
- **数分负责人**：@数据分析负责人
- **数仓负责人**：@数据仓库负责人

## 输出内容

生成的技术方案文档包含以下7个部分：

### 1. 项目概述
- 项目背景
- 指标清单（表格形式）
- 业务价值

### 2. 架构设计
- 整体架构图（文字描述）
- 数据流向
- 系统交互（表格）

### 3. 详细技术实现
为每个指标提供：
- 指标名称
- 业务口径
- 技术口径（完整SQL代码）
- 分子分母拆解
- 限制条件
- 数据来源（Aeolus任务ID/数据集ID）

### 4. 数据接入流程
- 接入步骤（5个阶段）
- 配置说明（表格）
- 验证方法（4项）

### 5. 质量保障
- 数据校验规则（4项）
- 监控告警（3类）
- 异常处理（3种场景）

### 6. 实施计划
- 阶段划分（表格）
- 时间节点（5个关键节点）

### 7. 风险评估
- 技术风险（表格，含应对措施）
- 业务风险（表格，含应对措施）

## 使用方法

### 方式一：通过sessions_spawn调用
```bash
sessions_spawn \
  runtime=subagent \
  mode=run \
  label="风神技术方案agent" \
  task="请处理以下需求：\n1. 截图路径：/path/to/screenshot.jpg\n2. 参考文档：https://bytedance.larkoffice.com/wiki/xxx\n3. 生成飞书文档发送给用户"
```

### 方式二：直接对话
向AI助手发送消息：
- 上传截图
- 提供参考文档链接
- 说明生成需求
- AI将自动调用风神技术方案agent生成文档

## 技术实现逻辑

### 第一步：解析截图
使用image工具提取表格信息：
- 业务线
- 指标名称
- 指标口径
- 业务方等字段

### 第二步：获取参考文档
使用feishu_fetch_doc获取参考文档内容：
- 数据来源（Aeolus任务ID/数据集ID）
- 技术口径示例
- SQL实现模板
- 限制条件

### 第三步：生成技术方案
根据提取的信息生成完整文档：
1. 按照参考文档的格式结构组织内容
2. 为每个指标编写完整的SQL代码
3. 基于参考实现调整技术口径
4. 生成完整的飞书Markdown文档

### 第四步：创建飞书文档
使用feishu_create_doc创建文档：
- 标题格式：风神Bubble指标接入技术方案_v1.0
- 内容：完整的Markdown格式
- 返回文档链接

## 注意事项

1. **数据源依赖**：确保Aeolus任务ID和数据集ID正确
2. **权限要求**：需要飞书文档创建权限（docs:document.media:upload）
3. **SQL准确性**：生成的SQL需要经过数仓团队review
4. **业务口径对齐**：上线前需与业务方确认口径理解一致
